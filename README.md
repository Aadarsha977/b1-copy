## Website

This is a standard Vite + React single-page app (SPA).

### Local development

1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build for production: `npm run build`
4. Preview production build: `npm run preview`

### Deploying to Vercel

- Build command: `npm run build`
- Output directory: `dist`

This repo includes a `vercel.json` rewrite so React Router routes work when you refresh or deep-link to a non-root URL.
